  3. 进入项目根目录执行npm install
  4. 项目打包npm run build

常用事件

  1. 播放
    播放分为开始播放和进入播放状态
    开始播放的动作是play
    进入播放状态的动作是playing
  2. 暂停
    暂停的动作是pause
  3. 跳转
    进入跳转动作是seeking
    跳转完成的动作是seeked

需要注意的是

  1. 视频从任何一个非播放状态进入播放状态，都需要经过playing的事件，如果需要配合视频播放做一些其他效果，只要监听好playing这个事件即可
  2. 我认为不是所有的事件都需要监听，监听对你有用的事件即可，对我来说，开始播放的事件比seeked更加重要，我只监听了seeking和playing
