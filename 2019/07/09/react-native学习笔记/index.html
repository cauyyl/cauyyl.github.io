Slow Animation

遇到的问题

Simulator突然变得特别的卡，动作特别的缓慢，backboardd的cpu占用率突然变得很高。

原因

因为不小心按到了快捷键Command+T，开启了慢动作模式，关闭这个模式就好了

说明

Slow Animation这个功能是慢动作，方便调试一些动画的，因为是刚接触ios APP开发，对simulatir的功能还不是很熟悉
